# IRC
port 8080 server 
```
make exe
```
On a different terminal : 
```
- nc -C <IP> <PORT>
- user 1 2 3 4 
- nick <Your_nickname>
```
Or
```
- irssi
- \connect <IP> <PORT>
```

You're now connected !!
Available commands : (add '\\' before the cmds on irssi)
```
- join <Channel_name> (add an # before the name if not on irssi)
- privmsg <dest>
- who <Channel>
- ping
```
